## 📋 Tentang Project

**AmikomEventHub** adalah aplikasi web untuk mengelola dan mempromosikan event kampus. Aplikasi ini menyediakan halaman publik untuk menjelajahi event yang tersedia, melakukan checkout tiket, serta panel admin yang lengkap untuk mengelola data event, kategori, dan transaksi.

### ✨ Fitur Utama

- 🏠 Halaman utama dengan daftar event
- 📄 Halaman detail event
- 🛒 Sistem checkout & pemesanan tiket
- 🎛️ Dashboard admin
- 📦 CRUD Event (Create, Read, Update, Delete)
- 🏷️ CRUD Kategori Event
- 💳 Manajemen transaksi
